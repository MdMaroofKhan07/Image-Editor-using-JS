# 🖼️ Image Editor

A modern and responsive **Image Editor Web Application** built using **HTML, CSS, and JavaScript**. This project allows users to import images, apply filters, rotate, flip, reset changes, and save the edited image directly to their device.

---

## 🚀 Features

- 📂 Import images from your device
- ✨ Apply image filters:
  - Brightness
  - Saturation
  - Inversion
  - Grayscale
- 🔄 Rotate image:
  - Rotate Left
  - Rotate Right
- ↔️ Flip image:
  - Horizontal Flip
  - Vertical Flip
- ♻️ Reset all applied filters and transformations
- 💾 Save edited image
- 🎨 Modern Neon Glassmorphism UI
- 📱 Responsive Design

## ⚙️ How It Works

### Import Image
Users can upload any image from their local system using the **Import Image** button.

### Apply Filters
Available filters include:

- Brightness
- Saturation
- Inversion
- Grayscale

The slider dynamically updates the selected filter value.

### Rotate & Flip

Users can:

- Rotate Left (90°)
- Rotate Right (90°)
- Flip Horizontally
- Flip Vertically

### Save Image

The edited image can be downloaded using the **Save Image** button.
